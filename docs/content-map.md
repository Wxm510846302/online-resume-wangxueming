# Word 简历到线上简历映射

来源文件：`/Users/xuemingwang/Downloads/blockchain-research/王学明开发岗个人简历v2.5.2.docx`

本项目不删除原简历内容，而是把内容拆成线上简历模块：

- 基本信息：映射到首页 hero、联系条和页面 SEO。
- 专业技能：映射到首页“能力矩阵”六个能力簇。
- 个人评价：映射到首页 summary、核心指标和“简历丰富建议”。
- 教育经历：映射到首页“教育与技术转型”模块。
- 工作经历：映射到“职业路径”时间线。
- 项目经验：映射到首页项目卡片和 `/project/:slug` 详情页。

项目详情页字段：

- `background`：业务背景和复杂度。
- `responsibilities`：候选人职责。
- `results`：结果指标和待补充量化项。
- `interviewAngles`：面试时建议重点讲法。
- `assets`：demo、视频、截图、架构图或待补充状态。

隐私提醒：

- 当前页面展示原简历手机号和邮箱。正式上线前建议确认是否需要手机号脱敏，或仅展示邮箱/二维码。
