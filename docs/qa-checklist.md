# QA Checklist

## 构建

- `npm run check:content`
- `npm run build`

## 视口

- 桌面：1440x900 首页、项目详情页。
- 平板：1024x768 首页。
- 手机：390x844 首页、项目详情页。
- 小屏：375x667 首页。

## 核心路径

- 首页导航可跳转能力、经历、项目、建议。
- 每个项目卡片进入详情页。
- 详情页刷新后仍可显示对应项目。
- 返回首页按钮可用。
- demo/视频/截图区域即使没有素材也有明确占位。
- 3D 人偶区域不遮挡正文，移动端不造成横向滚动。

## 内容

- 项目数量不少于原简历主要项目数量。
- 每个项目包含背景、职责、结果、面试讲法、素材建议。
- 技能矩阵覆盖 iOS、Flutter、小程序/Web、SDK、AI/Web3、技术管理。

## 可访问性

- 页面有唯一主标题。
- 按钮和链接可键盘聚焦。
- 图标按钮有 `aria-label`。
- 颜色对比度保持正文可读。
- 不出现空链接或空按钮。
